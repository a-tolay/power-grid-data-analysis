# German Power Grid (TSO) Data Analysis ⚡📊

## About The Project
This repository contains an Exploratory Data Analysis (EDA) of the electricity grid data from Germany's four major Transmission System Operators (TSOs): **50Hertz, Amprion, TenneT, and TransnetBW**. 

As an Electrical Engineering student, this project aims to combine my domain knowledge in power systems with data science skills. The analysis focuses on data preprocessing, time-series manipulation, and data visualization to extract meaningful insights from raw grid data.

## Technologies Built With
* **Language:** Python
* **Environment:** Jupyter Notebook
* **Libraries:** Pandas, NumPy, Matplotlib

## Dataset Information
The dataset consists of four distinct CSV files, each containing power generation/consumption data from a specific TSO in Germany:
* `50Hertz.csv`
* `Amprion.csv`
* `TenneTTSO.csv`
* `TransnetBW.csv`

## Key Processes & Learnings
* **Data Preprocessing:** Cleaning missing or inconsistent values and merging multiple CSV files into a unified dataset.
* **Time-Series Analysis:** Handling DateTime indexes to analyze temporal trends in power grids.
* **Data Visualization:** Creating clear graphical representations of electrical grid behaviors and data distributions.

## How to Run
1. Clone this repository to your local machine.
2. Ensure you have Python and Jupyter Notebook installed, along with the required libraries (`pandas`, `numpy`, `matplotlib`).
3. Run the `Pyhton_ile_Veri_Analizi_Odev_3(Jupyter).ipynb` file to view the analysis steps and outputs.
